在Windows上Vue-intro

2025/8/8 Vue 系列归纳 Vue 前端框架 JavaScript

Vue是一款流行的渐进式JavaScript框架，专注于构建用户界面。它的核心库只关注视图层，易于上手，适合构建单页应用（SPA）。本章节主要讲述在正式进入Vue实战内容之前关于Vue项目运行的地基。

1.Node.js基础

Node.js是什么?🤣

  • JavaScript运行时环境（运行js时不依赖浏览器）
  • JavaScript 本来是浏览器里的语言，负责网页上的交互，比如按钮点击、动画效果等。
    Node.js 让你能在浏览器外，也就是服务器端运行 JavaScript。

它基于 Chrome 浏览器的 V8 引擎（解析和执行 JS 的引擎），速度很快。 用 Node.js 可以做很多事情，比如搭建网站后台服务器、写命令行工具、处理文件、做网络请求等。
打个比方：以前你用 JavaScript 只能“给网页穿衣服”（做页面交互）有了 Node.js，你可以用 JavaScript“造房子、修路、开店铺”（做服务器、工具、应用程序）你如果想做网页后台开发、或者用 JS 写一些自动化脚本，Node.js 就非常重要。

Node.js具体能做什么?🤔

  • 运行构建工具(如webpack，vite)
  • 写后端服务(Express,Nest)
  • 管理本地工具(如Vue CLI)

🌼常用命令

  • node -v:查看node版本
  • node filename.js:运行js脚本
